Light Theramin

For this project, we create a light theramin that accepts an RGB value from three different phototransistors and outputs a value that is then used as the pitch the piezo plays.

Parts Required For Assembly

  • Power Cables
  • 1 Piezo
  • 2 phototransistors
  • 3 10kohm Resistors
  • 4 Small Jumper Wires
  • 4 Medium/Large Jumper Wire


const int redSensorPin = A0;
        const int blueSensorPin = A1;
        const int greenSensorPin = A2;
        
        int redSensorValue;
        int blueSensorValue;
        int greenSensorValue;
        int sensorValue;
        int sensorLow = 1023;
        int sensorHigh = 0;
        
        void setup() {
          // put your setup code here, to run once:
          Serial.begin(9600);
        }
        
        void loop() {
          // put your main code here, to run repeatedly:
          redSensorValue = analogRead(A0);
          blueSensorValue = analogRead(A1);
          greenSensorValue = analogRead(A2);
        
          sensorValue = (redSensorValue + greenSensorValue + blueSensorValue) / 3;
        
          int pitch = map(sensorValue, sensorLow, sensorHigh, 50, 4000);
        
          tone(8, pitch, 20);
        
          delay(10);
        }
