Spaceship Interface

For this project, we create a spaceship interface that has one LED light that is constantly on and blinks between two others when the button is pressed with an additional LED light that pulses regardless of if the button is pressed or not.

Parts Required For Assembly

  • Power Cables
  • 4 220ohm resistor
  • 1 10kilohm resistor
  • 2 Red LED Lights
  • 1 Green LED Light
  • 1 Blue LED Light
  • 1 Switch
  • 5 Medium/Large Jumper Wire


        int switchState = 0;
int pulseState = 0;

void setup() {
  // put your setup code here, to run once:
  pinMode(3, OUTPUT);
  pinMode(4, OUTPUT);
  pinMode(5, OUTPUT);
  pinMode(2, INPUT);
  pinMode(6, OUTPUT);
}

void loop() {
  // put your main code here, to run repeatedly:
  switchState = digitalRead(2);
  pulseState = digitalRead(6);

  if (switchState == LOW) {
    digitalWrite(3, LOW);
    digitalWrite(4, LOW);
    digitalWrite(5, HIGH);
  }


  else {
    digitalWrite(3, HIGH);
    digitalWrite(4, LOW);
    digitalWrite(5, LOW);

    delay(250);

    digitalWrite(3, LOW);
    digitalWrite(4, HIGH);
    delay(250);
  }

  if(pulseState == LOW) {
    delay(1000);
    digitalWrite(6, HIGH);
  }

  else if(pulseState == HIGH) {
    delay(1000);
    digitalWrite(6, LOW);
  }


}
